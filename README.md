# Void Blaster

Void Blaster is a desktop-first browser shooter built around a behind-the-ship flight view, neon tunnel traversal, modular combat systems, and a live-tuning rail for shaping feel in real time.

This repo is organized in the order we work:

1. `docs/reports` for audits, feasibility notes, and roadmap thinking
2. `docs/plans` for execution plans
3. `docs/operations` for standing rules, architecture, and team workflow
4. `app` for the playable game

## Current Stack

- `Vite`
- `TypeScript`
- `Three.js`
- `ESLint`

## Getting Started

```powershell
cd app
npm install
npm run dev
```
